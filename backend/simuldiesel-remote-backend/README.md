# Simuldiesel Remote Backend

Backend base do projeto Simuldiesel Remote Access, preparado para evolucao por etapas com Spring Boot e Maven.

## Objetivo

Estabelecer uma fundacao profissional para o backend, com organizacao de pacotes, configuracao inicial do build e documentacao tecnica para as proximas etapas.

## Stack

- Java 21
- Maven
- Spring Boot
- Spring Web
- Spring Data JPA
- Bean Validation
- H2 Database
- Spring Boot Test

## Estrutura Inicial

```text
backend/simuldiesel-remote-backend
├── docs
├── src/main/java/com/simuldiesel/remote
├── src/main/resources
└── src/test/java/com/simuldiesel/remote
```

## ETAPA 01

Esta etapa cria somente a fundacao estrutural do backend:

- projeto Maven com Spring Boot
- classe principal da aplicacao
- configuracao inicial em `application.properties`
- documentacao tecnica inicial
- organizacao de pacotes para futura expansao

Nao ha implementacao de regra de negocio, endpoints ou banco de dados nesta etapa.
